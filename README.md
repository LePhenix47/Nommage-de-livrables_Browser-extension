## Purpose

The purpose of this web extension is to name the deliverables for the website [OpenClassrooms](www.openclassrooms.com)

*Example:*

- Before → "Nom_Prénom_1_code_mmaaaa"

- After → "Younes_Lahouiti_1_code_012023"
